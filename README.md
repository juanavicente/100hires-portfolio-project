## Issues Encountered and How I Solved Them

### Finding the Required Extensions

One of the instructions was to install the Claude Code and Codex extensions inside Cursor.

Initially, I was unable to find them in the marketplace. I searched using different keywords, explored the available options, and compared what I was seeing with the instructions provided.

Instead of assuming something was wrong, I continued investigating and verified whether I was using the latest version of Cursor. After updating the application and reopening the project, both extensions became available and I was able to install them successfully.

### Learning a New Environment

Although I was already comfortable with Git and GitHub, Cursor was completely new to me.

Some menus and workflows were different from what I expected, so I spent time exploring the interface, understanding how repositories are managed, and becoming familiar with the available AI features.

I solved this by experimenting, reading available documentation, and learning through practice while completing the assignment.
